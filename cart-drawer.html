Customizations on cart drawer

Cart drawer type

The Prestige theme already has a cart drawer feature.

This functionality was further extended by adding a discount code checker on the cart itself.

How it works

A separate JS file is called and it contains JSON data of all the discount codes and the discount details on the store.
This file gets updated by a Google Cloud Server.
This server retrieves the data via API processes and then logs them in the JS file.

All important data needed for calculations and checking are in the JS file:

  • discount code valid
  • accepted products
  • discount type
  • discount value
  • discount minimum price
  • number of uses

With these data, the line items can be easily manipulated for every cart change.

On click of the checkout button, the discount code is added to the checkout url so it will be automatically applied on the checkout page.

The discount code last used is also stored in a cookie - so the cart automatically applies it every time the customer access the store.
