Shopify Scripts

Was sind eigentlich Shopify Scripts?
Es sind kleine Customizations, die es ermöglichen die komplexen Anforderungen, die E-Commerce heutzutage stellt abzudecken.
Diese Scripts müssen von einem erfahrenen Shopify Developer geschrieben werden, da sie mithilfe der Ruby API geschrieben werden.
Der Code läuft auf Shopify Servern. Man hat also keine Probleme mit Third-Party Support.

Was kann beispielsweise mit Shopify Scripts umgesetzt werden:

  • Eine "kaufe 3, zahl 2" Aktion
  • Rabatt für Produkte mit bestimmten Tag
  • Ahnzahlbegrenzung für ein Produkt
  • Verändere, Verberge, Ordne Versandoptionen

Welche Shopify Scripts gibt es

Es gibt verschiedene Arten von Scripts:

  1. Line Item Script: Sie beeinflussen die Objekte im Cart. Preise können verändert werden, Rabatte können gewährt werden, Quantity kann verändert werden. Das Skript läuft jedesmal, wenn ein Objekt zum Cart hinzugefügt, entfernt, oder geändert wird.
  2. Shipping Script: Versandmethoden können verändert werden und auf bestimmte Versandarten können Rabatte gewährt werden.
  3. Payment Script: Umbenennen, Verstecken oder neu Ordnen von Bezahlmethoden.

Beschränkungen

Mit Shopify Scripts sind allerdings meist nur bestimmte vordefinierte Sachen umsetzbar.
Durch die Script API ist man limitiert. Es ist also auch nur das umsetzbar, was von Shopify vorgegeben wird.

Ein weiterer Nachteil ist die Berechnungsbegrenzung, die vorgegeben wird. Wenn man sehr viele Anpassungen vornimmt, dann wird das Limit erreicht und das Skript wirft Fehler, da das Berechnungslimit überschritten wurde.
Im allgemeinen ist der Entwicklungsprozess mit Shopify Scripts schwer, da man keine Entwicklungsumgebung hat in der der Code debuggt werden kann.

Warum Du trotzdem irgendwann Shopify Scripts verwenden wirst

Ein sehr schönes Beispiel die Anzahlbegrenzung bei E-Books. Ein E-Book ist ein digitales Produkt. Es macht für einen Kunden meist keinen Sinn das Produkt zweimal zu kaufen. Eine Anzahlbegrenzung auf 1 im Warenkorb ist daher sinnvoll.
Auch wenn du zum Beispiel ein Produkt anbietest, dass kostenlos ist (Geschenk jeglicher Art). Hier müssen sogar Shopify Scripts verwendet werden, es wäre katastrophal, wenn ein Kunde unbegrenzt kostenlose Produkte bestellen könnte.
Du solltest es aber nicht übertreiben und mehrere Aktion wie "Kaufe 3, zahle 2" gleichzeitig über Scripts wegen oben genannter Problematik laufen lassen. Vergewissere Dich am besten mit Hilfe eines erfahrenen Developer, dass alles problemfrei funktioniert.
