Theme Transfer

Prestige theme transfer

For a faster store experience, it was decided to switch to Shopify's Prestige theme.

Switching to a new theme is basically moving content (added images, copy, etc.) and customizations (settings, modifications, etc.) to a different theme.
This is a manual process involving two separate theme files: the original theme with content and the new theme without content.

Step 1: Duplicate theme

It is recommended to make a back-up of the current live theme.
To do that, in the Themes page, click on the three dots icon for your published theme, and click on "Duplicate"

Step 2: Setup

Setup on theme editor the sections of the current live theme pages to the new theme.
As long as there is a section that looks like the current one on the new theme, set it up first and do customizations later.

Step 3: Customizations

Modify ezisting sections of the new theme to match the current one. Most of the time, you will need to code new section completely.

Make sure custom features present on the current theme will be transferred to the new theme.
This includes custom coded functionalies.

Double-check apps if they work flawlessly on the new theme.

Step 4: Publish

Test the new theme extensively and make sure everything works as expected.

It is best to not delete the old theme so you can always go back to it if something unexpected happpens.
